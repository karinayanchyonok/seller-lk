# Запуск проекта поэтапно

### Перейти в папку с сервером

```bash
cd server
```

### Установить зависимости

```bash
 npm i
```

### Запустить сервер

```bash
PORT=8081 node index.js
```

### Удостоверьтесь, что порты совпадают с данной строкой в src/api/client.ts

```ts
const API_BASE_URL = 'http://localhost:8081';
```

###  Перейдите в папку

```bash
cd ../src
```

### Установить зависимости

```bash
 npm i
```

### Запустите фронтенд

```bash
 npm run dev
```

### Перейдите на данную страницу в браузере

```
 http://localhost:5173/
```

Зарегистрируйтесь на openrouter.ai

Перейдите в раздел API Keys

Нажмите Create Key

Скопируйте ключ

Вставьте в .env файл
