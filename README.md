# 1. Как запустить проект?

## Запустить бэкенд

### Перейти в папку с сервером

```bash
cd server
```

### Установить зависимости

```bash
 npm i
```

### Запустить сервер

```bash
PORT=8080 node index.js
```

### Удостоверьтесь, что порты совпадают с данной строкой в src/api/client.ts

```ts
const API_BASE_URL = 'http://localhost:8081';
```

## Подключение нейронки
Зарегистрируйтесь на ```openrouter.ai```

Перейдите в раздел API Keys

Нажмите Create Key

Скопируйте ключ

Создайте .env файл по .env.example

Вставьте в .env файл свой апи ключ в VITE_OPENROUTER_API_KEY, остальные не нужно менять

## Запуск фронтенда

###  Перейдите в папку

```bash
cd ../src
```

### Установить зависимости

```bash
 npm i
```

### Запустите фронтенд

```bash
 npm run dev
```

### Перейдите на данную страницу в браузере

```
 http://localhost:5173/
```
